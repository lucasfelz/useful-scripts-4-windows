## Overview

This repository contains a curated collection of PowerShell and Batch scripts designed for:
- **System Administration** - Automate common Windows tasks
- **Reconnaissance** - Information gathering on Windows systems
- **Penetration Testing** - Security assessment utilities
- **Post-Exploitation** - Privilege escalation and persistence

## Installation

### Clone the repository
```bash
git clone https://github.com/yourusername/useful-scripts-4-windows.git
cd useful-scripts-4-windows
```

### Set Execution Policy (PowerShell)
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### Verify Installation
```powershell
Get-ChildItem -Recurse *.ps1 | Measure-Object
```
